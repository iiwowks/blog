## 仓库的UI界面

1. watch 的主要功能：
   - watch 一个仓库，以后该仓库的更新信息就会显示在用户的公开活动中。
2. gist 的主要功能：
   - 管理及发布一些没有必要保存在仓库中的代码，例如一些小的代码片段
3. wiki 的主要功能：
   - 常用于记录开发者之间应该共享的信息或者软件文档，数字代表当前Wiki的页面数量
4. Graphs : 以图表的形式显示仓库的各项指标
