## 全局配置

```
git config --global user.name 'name'
git config --global user.email 'email'
git config --global --list
```

## 建git仓库

1. 把已有项目代码纳入git管理

```
cd folder
git init
```

1. 新建项目直接用代码管理

```
cd 某个文件夹
git init your_project
cd your_project
```

工作目录->(git add files)暂存区->(git commit)版本历史
