## IDEA

### 快捷键

* 系统：
  * `ctrl + shift + a`: 快速查找功能
  * `alt + enter`: 提示
  * `ctrl + alt + s`: setting面板

  * `shift + ctrl+alt+u`:   各种类，接口及继承图，太漂亮了！
* 模板：  
  * `psvm`: 生成`public static void main(String[] args){}`
  * `sout`: 生成`System.out.println();`
  * `fori`: for循环
  * `itli`: List迭代
* 编辑：
  * `ctrl + r`: 替换关键字
  * `ctrl + shift + f`: 全局查找关键字
  * `ctrl + shift + n`: 全局查找文件
  * `ctrl + alt + shift + j`: 同名单词列操作
  * `ctrl + w`: 选中单词
  * `ctrl + shift + enter`: 自动回车加`;`号
  * `shift + enter`: 自动回车
  * `shift + F6`: 重命名
* 跳转：
  * `ctrl + g`: 跳转到指定行号
  * `ctrl + shift + 1~9`：添加页签
  * `ctrl + 1~9`: 跳转页签

### java 不支持发行版本xxx

maven项目：Error : java 不支持发行版本5

1. 配置一
在 Setting->Java Compiler 中将 Project bytecode version 与 target bytecode version 全部设为与本地版本相同的 version 13

2. 配置二
在 Project Structure->Modules->Language level 也设置成 13，与上面配置的版本一致

## mysql workbench

### 快捷键

* 新建tab(new tab) `ctrl+t`
* 执行当前语句(execute current statement) `ctrl+enter`
* 执行全部或选中的语句(execute all or selection) `ctrl+shift+enter`
* 查看执行计划(explain current statement) `ctrl+alt+x`

## g.itignore语法规则：

* 以【/】表示目录以【*】匹配多个字符以【?】匹配单个字符
* 以【[]】匹配字符列
* 以【!】忽略文件或者目录
