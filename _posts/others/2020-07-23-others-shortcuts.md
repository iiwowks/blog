## vscode快捷键

| 快捷键                | 功能                                           |
| --------------------- | ---------------------------------------------- |
| `ctrl + 左右`         | 选单词                                         |
| `ctrl + shift + 左右` | 选整行                                         |
| `输入/**+enter`       | 方法说明注释，自动根据参数和返回值生成注释模板 |

## idea快捷键

| 快捷键                   | 功能                                           |
| ------------------------ | ---------------------------------------------- |
| `ctrl + shift + a`       | 快速查找功能                                   |
| `alt + enter`            | 提示                                           |
| `ctrl + alt + s`         | setting面板                                    |
| `shift + ctrl+alt+u`     | 各种类，接口及继承图，太漂亮了！               |
| `psvm`                   | 生成`public static void main(String[] args){}` |
| `sout`                   | 生成`System.out.println();`                    |
| `fori`                   | for循环                                        |
| `itli`                   | List迭代                                       |
| `ctrl + r`               | 替换关键字                                     |
| `ctrl + shift + f`       | 全局查找关键字                                 |
| `ctrl + shift + n`       | 全局查找文件                                   |
| `ctrl + alt + shift + j` | 同名单词列操作                                 |
| `ctrl + w`               | 选中单词                                       |
| `ctrl + shift + enter`   | 自动回车加`;`号                                |
| `shift + enter`          | 自动回车                                       |
| `shift + F6`             | 重命名                                         |
| `ctrl + g`               | 跳转到指定行号                                 |
| `ctrl + shift + 1~9`     | 添加页签                                       |
| `ctrl + 1~9`             | 跳转页签                                       |
