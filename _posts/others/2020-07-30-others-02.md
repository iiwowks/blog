* `java 不支持发行版本xxx`: maven项目：Error : java 不支持发行版本5
  * 配置一:在 Setting->Java Compiler 中将 Project bytecode version 与 target bytecode version 全部设为与本地版本相同的 version 13
  * 配置二:在 Project Structure->Modules->Language level 也设置成 13，与上面配置的版本一致
