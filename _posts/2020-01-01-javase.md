# javaSE foundation

## 基本语法

- 数据类型：
  - 整数：
    - `byte` : 1
    - `short` : 2
    - `int` : 4
    - `long` : 8
    - `0b` : 2 进制
    - `0x` : 16 进制
    - `0` : 8 进制
  - 浮点数：
    - `float` : 4
    - `double` : 8
    - `BigDecimal` : 大数
  - 字符：
    - `char` : 2
    - `ascii`
    - `utf-8`
    - `Unicode`
    - `\u0000`: `\b \n \r \n \t \' \" \\`
  - 布尔值： `boolean` : 1
- 引用数据类型：
  - 类
  - 接口
  - 数组
- 类型转换：
  - 自动类型转换：低转高
  - 强制类型转换：高转低
- 变量和常量：
  - type varName
  - 作用域：
    - 类变量
    - 实例变量
    - 局部变量
  - 常量 `final MAX_A = 10`
  - 命名规范：
    - 见名知意
    - 驼峰命名(变量，方法)
    - 类首字母大写，驼峰命名)
    - 常量：大写 + 下划线
- 运算符：
  - 算术运算符 `+ - * / % ++ --`
  - 赋值运算符 `=`
  - 关系运算符 `> < >= <= == != instanceof`
  - 逻辑运算符 `&& || !`
  - 位运算符 `& | ~ ^`
  - 条件运算符：`? :`
  - 拓展运算符：`+= -= *= /=`
- 包机制
  - 域名倒写
  - 防止命名冲突
  - package
  - import
- **JavaDoc**

  - `@author`
  - `@Version`
  - `@Since`
  - `@param`
  - `@return`
  - `@throws`
  - 生成文档：`javadoc -encoding UTF-8 -charset UTF-8 HelloWorld.java`

## 流程控制

- 顺序 ：自上而下
- 选择 结构：`if, if-else, switch`
- 循环 结构: `while, do-while, for, for each`
- `break & continue`：
  - `break` 跳出循环
  - `continue` 终止当次循环
  - 带标签`continue`
  - `return`

## 方法

- 定义：修饰符 返回值 方法名 (参数名) {return 返回值;}
- 调用：
  - 类名.方法
  - 对象.方法
- 方法重载： 名字相同，参数列表不同
- 命令行传参： 给 main 方法传参
- 可变长参数：
  - `...`
  - 必须放在最后一个参数
- 递归：
  - 出口

## 数组

- 定义：`new int[5]`
  - `{1, 2, 3}`
  - 必须是同一类型
- 二维数组： `int[][]`
- `Arrays`工具类

## 面向对象

- 类与对象：
  - 类是对象的抽象：模板`class`
  - 对象是类的具体
- 构造方法：
  - 构造重载：
    - 默认的无参构造
    - 如果手动定义了有参构造，手动加一个无参构造
    - 单例模式，需要构造器私有
- `new`对象：
  - 栈存放引用
  - 堆存放引用
- 封装：属性私有， `get`, `set`
- 继承：
  - `extends`
  - `Object`
  - 子类拥有夫类所有特性
  - 方法重写
  - `this`
  - `super`
  - Java 是单继承，只能继承一个父类
- 多态：
  - 父类的引用指向子类的对象： `Person person = new Student()`
  - `instanceof`如果匹配，可以进行类型转换
- 修饰符：
  - `public`
  - `private`
  - `protected`
  - `static`
  - `final`
  - `abstract`
- 接口：
  - `interface`：
    - 只能定义方法名
    - 子类实现接口，必须重写其中的方法
    - 只有一个方法的接口，叫做**函数式接口**，可以使用**lambda 表达式**简化
    - 接口比抽象类更抽象
    - 一个类可以实现多个接口
- 内部类：
  - 局部内部类
  - 静态内部类
  - 匿名内部类
