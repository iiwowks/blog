## 2.1 语法

<br>
<br>
<br>

### 2.1.1 插值

{%raw%}
```
1. 文本插值使用双大括号{{}}
<span>Text: {{ text }} </span>
2. 只需要渲染一次数据使用 * 实现
<span>Text: {{ *text }} </span>
3. 双大括号标签把里面所有的值当作字符串处理，值是html片段使用三个大括号绑定
<div>Logo: {{{ logo }}} </div>
logo: '<span> DDFE </span>'
4. 放在html标签中
<li data-id = '{{ id }}'></li>
```
{%endraw%}

### 2.1.2 表达式

{%raw%}
```
{{ cents/100 }}
{{ true ? 1 : 0 }}
{{example | filterA | filterB}} 在表达式后面添加过滤器
{{example | filter a b}} 过滤器传入参数，a参数和b参数用空格隔开
```
{%endraw%}

### 2.1.3 指令

指的是带有v-前缀的特殊特性，其值限定为绑定表达式。  
作用： 当表达式的值发生变化时，将这个变化反映到DOM上。

## 分隔符

1. `delimiters` ：` Vue.config.delimiters = ["<%", "%>"]`修改默认的文本插值分隔符，文本插值语法变成`<%example%>`
2. `unsafeDelimiters`：`Vue.config.unsafeDelimiters = ["<$", "$>"]`， 修改默认的html插值的分隔符，html插值语法变成`<$example$>`
