7. 在使用 Jekyll 博客时，发现使用双括号$\lbrace\lbrace\rbrace\rbrace$或者$\lbrace\% \%\rbrace$等`liquid tags`，页面会自动渲染这些标签。然而我需要的是在 markdown 中显示这些标签的字符，而不是被解析渲染。<mark style="background: green">jekyll 博客中是无法显示 liquid 标签的!</mark> - 解决方法一：<mark>使用 liquid 的 raw 标签</mark>：$\lbrace\% raw \%\rbrace$ 和 $\lbrace\% endraw \%\rbrace$包裹代码块。 - 解决方法二：使用页面变量，创建一个页面变量$lcb: "\lbrace"$，然后代替标签中的左标签： $\lbrace\lbrace lcb \rbrace \rbrace \% endfor \%\rbrace$
   示例：
   {% raw %}

```html
<ul>
  {% for tag in site.tags %} {% assign name = tag | first %} {% assign posts =
  tag | last %}
  <li>{{ name | camelize | replace: "-", " " }} has {{posts | size}} posts</li>
  {% endfor %}
</ul>
```

{% endraw %}
