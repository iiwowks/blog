> 官方文档：

## 环境搭建

* 浏览器：`chrome`
* IDE: `Visual Studio Code`
* `Node.js 8.9+, npm`

## 安装

### 直接用`<script>`引入

```
<script src="https://cdn.jsdelivr.net/npm/vue@2.6.11"></script>
```

### 命令行工具cli

```
安装：
npm install -g @vue/cli
# OR
yarn global add @vue/cli\
```
```
创建一个项目：
vue create my-project
# OR
vue ui
```

## 快捷键

`输入vue+tab键`: 模板快速生成
`ctrl+alt+L`: 代码格式化

## 本地运行

```
npm run serve
```
